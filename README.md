## 依赖下载

```
使用npm 或者 yarn
cnpm 诡异bug多
```

# 解决 npm 下载慢

```sh
npm install
npm install --registry=https://registry.npmmirror.com
```

### 启动环境

```sh
npm run dev
npm run pro
npm run test
```

### 环境打包

```sh
npm run build
npm run build:pro
npm run build:test
```

### 生产模拟

```sh
npm run preview
```

### Lint with [ESLint](https://eslint.nodejs.cn/docs/latest/rules/)

# 查看是否符合规则

```sh
npm run lint
```

### 规则配置选项 [prettier](https://prettier.io/docs/en/options)

# 格式化代码风格

```sh
npm run format
```
